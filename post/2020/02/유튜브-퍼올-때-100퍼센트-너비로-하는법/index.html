        유튜브 너비 100%로 퍼오기

        k1005
        2020-02-23
        • YouTube
        • 블로그 노하우
        • CSS 활용

        유튜브 영상을 블로그나 페이지에 삽입시 고정 너비로 삽입하기 싫을 경우가 있다. 이럴때는 몇가지 @media 사이즈를 정해서 할 수도 있지만 너비가 유동적일 때는 width 값을 100%로 주고 높이를 auto로 해주고 싶다.

        이럴때 사용 하는 팁! 유튜브 영상 자동으로 높이조절하기!

        # 1. css 코드를 추가한다

        유튜브를 화면에 꽉 채우기 위해 미리 CSS 작업을 해 두어야 한다. 자신이 관리 운영 하는 사이트라면 문제 없겠지만 보통은 티스토리나 네이버블로그와 같은 남의 회사 서버에 블로그를 쓰고 있을 테니 그럴 때는 어떻게 수정해야 하는지도 함께 알아보자.

        # 우리가 입력해야 할 CSS 코드

        이미 CSS 수정하는 법을 알고 있는 사람도 많을 것으로 생각된다. 그런 분들은 바로 알아서 입력 해주시고 다음 순서로 넘어가시면 되겠다.

        /* 유튜브 width 100 퍼센트로 */
.video-container {
  position: relative;
  padding-bottom: 56.25%;
  padding-top: 30px;
  height: 0;
  overflow: hidden;
}
.video-container iframe,
.video-container object,
.video-container embed {
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
}

        # 티스토리 CSS 추가하기

        우선 티스토리에 자신이 블로그를 운영하고 있는 계정으로 접속해서 로그인 한다. 그리고 다음 부터 보여지는 그림과 똑같이 생긴 메뉴를 찾아 가기면 하면 된다. 순서는 다음과 같다.

        1. 스킨편집
        2. html편집
        3. CSS 수정메뉴
        4. CSS 입력 후 저장
        5. 2번 그림에 보이는 위에 적용 버튼 누르면 끝

        # 네이버 블로그 CSS 추가하기

        네이버는 블로그가 없기 때문에 패스 하도록 하겠습니다. (급 공손 모드) 나중에 네이버에 블로그 개설하게 되면 수정하면서 업데이트 할게요.

        # 2. 유튜브 영상을 퍼와서 코드 수정까지

        유튜브는 자체적으로 영상공유 기능을 갖고 사람들이 활발하게 퍼갈 수 있기를 원한다. 그렇기 때문에 우리는 아주 몇번의 움직임 만으로 유튜브 영상을 퍼올 수 있다.

        1. 유튜브 접속
        2. 원하는 영상을 찾아 시청화면으로
        3. 그림따라 코드 복사 해서 코드 붙여 넣기

        이번에도 티스토리 사용자를 예로 들어 그림을 준비했다.

        # 3. HTML 코드 수정 예시

        html 코드에서 iframe을 <div class="video-container"></div>로 감싸준다

        <div class="video-container">
  <iframe width="560" height="315" src="https://www.youtube.com/embed/wzp0SWXS-zk" frameborder="0" allow="accelerometer; autoplay; encrypted-media; gyroscope; picture-in-picture" allowfullscreen></iframe>
</div>
